容器

用于布局的容器组件，方便快速搭建页面的基本结构：

组件采用了 flex 布局，使用前请确定目标浏览器是否兼容

基础用法

使用单一分栏创建基础的栅格布局。

分栏间隔

使用 gutter 属性来指定每一栏之间的间隔，默认间隔为 0。

分栏偏移

通过制定网格属性可以指定分栏偏移的栏数。

参数 说明 类型 可选值 默认值

scalable 是否允许扩展 boolean - -

rows 行数 Array / Number - -
cols 列数 Array / Number - -
gutter 元素之间的间隔 Number - -
cells 格子描述 Object - -
height 高度 String - -
$body 子节点 Array - -
