标签页

分隔内容上有关联但属于不同类别的数据集合。

基础用法

基础的、简洁的标签页。

选项卡样式

选项卡样式的标签页，将mode设置为card。

卡片化

卡片化的标签页，将mode设置为border-card。

默认选中

通过设置defaultActive，默认选择打开标签。

Props

参数 说明 类型 可选值 默认值
mode 风格类型 string card/border-card -
tabs 标签页 tabsProps - -
defaultActive 默认选中 String - -

tabsProps

参数 说明 类型 可选值 默认值
name 标签页id string - 'tab' + Date.now()
title 标签名 string - -
content 标签子节点 Array - -
