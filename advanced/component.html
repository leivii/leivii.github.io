注册组件

Leivii编辑器 从底层设计上支持扩展，实际上 Leivii 本身的组件也是通过注册机制添加进去的。

注册组件的方法

Leivii.API.registComponent(name?: string, component: Vue)
1

例如我们有以下组件：

<template>
  <div class="my-card" :class="$classes">
    <div class="card-header">
      <div class="title">{{ $transMsg(title) }}</div>
      <div class="actions">
        <el-button size="small" @click="$emit('add')">添加</el-button>
      </div>
    </div>
    <div class="geek-card-body">
      <slot v-if="!$body || !$body.length" />
      <leivii :data="$body" />
    </div>
  </div>
</template>

<script>

export default {
  name: 'MyCard',
  options: {
    name: '卡片',
    parents: ['all'], // all
    groups: ['container'],
    events: {
      add: '添加'
    }
  },
  props: {
    title: {
      type: String,
      clazz: 'I18n',
      placeholder: '请输入标题',
      label: '标题'
    }
  }
}
</script>

<style scoped>
.my-card {

}

  align-items: center;

  font-weight: bold;
  padding: 8px 15px;

}
</style>
